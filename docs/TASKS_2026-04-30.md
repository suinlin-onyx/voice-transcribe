# 转录服务优化任务

## 目标

优化转录服务的输出质量，解决以下问题：
1. 换行逻辑不统一
2. 流式输出未累积
3. 非语音内容未过滤
4. 断句不合理

### T4: 非语音内容过滤 ✅

**问题**：键盘声、打字声被误识别

**修改**：
- [x] 后处理过滤单字/无意义内容
- [ ] 调整 VAD 参数（待定）

**文件**：
- `processors/asr_processor.py` ✅ 已实现
- `transcribe_server_v3.py`（调用）
- `docs/VAD_FILTER_2026-04-30.md`（已创建）

**过滤规则**：
```python
# 过滤纯标点符号
if re.match(r'^[\.\,\!\?\。\，\！\？\s]+$', text):
    return ""

# 过滤单字（中文单字或英文单词太短）
if len(text.strip()) <= 1:
    return ""

# 过滤无意义内容（全是字母且太短）
if re.match(r'^[a-zA-Z\s]+$', text) and len(text.strip()) < 3:
    return ""

# 过滤纯数字
if re.match(r'^[\d\s\.\,\-]+$', text.strip()):
    return ""
```
