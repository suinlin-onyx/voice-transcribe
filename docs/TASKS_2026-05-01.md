## 二、架构模块划分

```
┌─────────────────────────────────────────────────────────────┐
│                    API Server (端口 8080)                    │
│  - WebSocket 连接管理                                        │
│  - HTTP REST API (/health, /control)                       │
│  - 消息协议转换                                             │
│  - 状态主动推送                                            │
└─────────────────────┬───────────────────────────────────────┘
                      │ asyncio.Queue
┌─────────────────────▼───────────────────────────────────────┐
│                  Model Server                               │
│  - ASR/VAD 推理 (SenseVoiceSmall)                         │
│  - 模型加载/卸载                                           │
│  - 文字输出                                                │
└─────────────────────┬───────────────────────────────────────┘
                      │ asyncio.Queue
┌─────────────────────▼───────────────────────────────────────┐
│                  Text Processor                             │
│  - 筛选 (噪声过滤)                                         │
│  - 拼接 (分段文字合并)                                      │
│  - 格式化 (标点、换行)                                     │
│  - 数字修复 (6.3 → 6.3)                                   │
│  - 英文空格修复 (p p t → ppt)                              │
└─────────────────────────────────────────────────────────────┘
```

## 四、重启脚本

### 4.1 热重启 TextProcessor

```bash
curl -X POST http://127.0.0.1:8080/control/reload-text-processor
```

### 4.2 热重启热词

```bash
curl -X POST http://127.0.0.1:8080/control/reload-hotwords
```

### 4.3 重启 Model Server (慎用)

```bash
curl -X POST http://127.0.0.1:8080/control/restart-model
```

---

## 五、重启代价对比

| 操作 | 耗时 | 影响范围 |
|------|------|----------|
| TextProcessor reload | ~1s | 仅当前任务 |
| 热词 reload | ~2s | 无 |
| Model Server restart | **~30s** | 所有推理中断 |

## 六、接口清单

### 6.1 HTTP 接口

| 端点 | 方法 | 说明 |
|------|------|------|
| `/health` | GET | 服务健康检查 |
| `/status` | GET | 服务状态查询 |
| `/control/reload-text-processor` | POST | 热重启 TextProcessor |
| `/control/reload-hotwords` | POST | 热重载热词 |
| `/control/restart-model` | POST | 重启模型服务 |

### 6.2 WebSocket 消息

```json

{
  "action": "reload_text_processor"
}

// 服务端 → 客户端
{
  "status": "ok",
  "message": "TextProcessor reloaded"
}
```
